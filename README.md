## 基于react native 技术栈的一款类似于滴滴的页面的导航软件（内嵌高德地图，七陌客服原生sdk）
### 涉及技术
* router: react-native-router-flux
* 字体图标：react-native-vector-icons
* UI库：Antdesign 
* store:mobx
* 高德地图：react-native-amap3d
* 数据缓存方案: AsyncStorage
* 客服：七陌客服原生sdk
# Installation【安装】
在根目录执行
```
yarn && cd ios && pod install && cd ..
```
运行
```
yarn ios || yarn android
```
